== Installation ==
1. Upload `add-rows-125` folder to the `/wp-content/plugins/` directory or install automatically through your site.
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Settings > Ad Rows 125 and fill out the ad details
4. Go to Widgets and ad the widget to an area

== Frequently Asked Questions ==
